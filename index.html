HTML

  • HTML building blocks, CSS to define style and JavaScript to make the website interactive
  • = element contains information, or Metadata, that helps the browser to render the page correctly. Elements within the are not visible when a user looks at the webpage
  • Inside element, we can use the following Metadata tags:
  • -Style: element defines the CSS styles associated with a website
  • -Base: element defines the base URL for a webpage
  • -Link: element connects an external resource to the HTML document
  • -Meta: element defines metadata such as the character set, description, keywords, author, viewport
  • -Script: element embeds client-side scripts in the HTML document
    • <= element contains the content that is shown to the user
